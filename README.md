# Taskflow Project

Aplicación web de gestión de tareas desarrollada con:
- HTML
- CSS
- JavaScript
- LocalStorage
- Tailwind CSS

## Funcionalidades
- Añadir tareas
- Eliminar tareas
- Guardado automático con LocalStorage
- Modo claro / modo oscuro
- Interfaz responsive
